# Global-Literacy-Education-Trend-Analysis
this project analyzes global trends in literacy, education, and economic indicators using data from multiple sources. The data is cleaned, transformed, and explored using Python, SQL, and visualization tools.

Key features of the project include:

📌 Data cleaning and preprocessing using Pandas
📊 Exploratory Data Analysis (EDA) with multiple visualizations
🗄️ SQL queries to extract meaningful insights
🌐 Interactive Streamlit app with:
SQL Query Executor
EDA Visualizations
Country-wise analysis

Tools & Technologies:

Python (Pandas, Matplotlib, Seaborn)
SQLite
Streamlit
Power BI (for dashboards)

This project helps understand the relationship between literacy, education, and economic growth across countries.
